# GenAINet

GenAINet is a .NET solution dedicated to exploring and implementing Generative AI capabilities.

## Projects

### GenAINet.TextGen

A console application built with .NET 9.0 focusing on text generation features.

## Getting Started

### Prerequisites

- [.NET 9.0 SDK](https://dotnet.microsoft.com/en-us/download/dotnet/9.0)

### Building the Solution

To build the entire solution, run dependencies:

```bash
dotnet build
```

### Running the Application

To run the text generation console app:

```bash
dotnet run --project src/Consoles/GenAINet.TextGen/GenAINet.TextGen.csproj
```

## Structure

- `src/Consoles/GenAINet.TextGen`: Console application for text generation experiments.
- `GenAINet.sln`: The main solution file.
